# Theme plugins

Drop `*.theme.json` files here and register them in `app/static/plugins/index.json` under the `themes` array.

Built-in themes (beta4.0):
- `ocean-public.theme.json`
- `black-green.theme.json` (beta1.1 default dark green style with pure black background image)
- `aurora-neon.theme.json` (new in beta4.0)
- `sand-dune.theme.json` (new in beta4.0)
